To use this script. Enable Beta APIs. 
Make sure you have operator permission. if you do, you can run the code by typing !run in the chat.

To write the code, use a book and quill.

# The language currently has:
1. say() logs the argument.
2. readNext() reads the next thing user types in chat.
3. Variables:
`steve x = 10`
5. if...else:
```
if (condition)
--code--
else
--code--
end
```
Nesting is SUPPORTED.

6. Functions:
```
fn name (argu, ments)
steve x = /argu/ + /ments/
fnEn
```

7. Loops:
```
repeat (10)
--code--
rEnd
```

Nesting is SUPPORTED.

8. Basic arithmetic:
Only one operation at a time.
Available operations:
+, -, /, *.
